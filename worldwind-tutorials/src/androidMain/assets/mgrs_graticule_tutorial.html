MGRS Graticule Tutorial

Demonstrates the MGRS Graticule capability.

This example adds a layer to the basic globe that shows the MGRS Graticule.

Example

MGRSGraticuleFragment.kt

The MGRSGraticuleFragment class extends the BasicGlobeFragment and overrides the createWorldWindow method. Here we simply add an additional MGRSGraticuleLayer to the globe.
