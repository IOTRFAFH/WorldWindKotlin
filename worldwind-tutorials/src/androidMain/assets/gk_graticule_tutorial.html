Gauss-Kruger Graticule Tutorial

Demonstrates the Gauss-Kruger Graticule capability.

This example adds a layer to the basic globe that shows the Gauss-Kruger Graticule.

Example

GKGraticuleFragment.kt

The GKGraticuleFragment class extends the BasicGlobeFragment and overrides the createWorldWindow method. Here we simply add an additional GKGraticuleLayer to the globe.
